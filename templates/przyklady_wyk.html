Przykłady

Przykład 1.

Wyznacz \(x\) spełniające równanie

\[\left(\sqrt[3]{2} \right)^{x-3} = 16 ^ {\frac{1}{3}\, x+2}\, .\]

Rozwiązanie

\[\begin{aligned} \left(\sqrt[3]{2} \right)^{x-3} &= 16 ^ {\frac{1}{3}\, x+2} \\[10pt] 2^{\frac{1}{3}\, x-1} &= 2^{\frac{4}{3}\, x+8} \\[10pt] \text{funkcja jest różnowarto} &\text{ściowa, zatem otrzymujemy}\\[10pt] \frac{1}{3}\, x-1 &= \frac{4}{3}\, x+8 \\[10pt] x &= -9 \end{aligned}\]

Odpowiedź: \(x = -9\).



Przykład 2.

Wyznacz \(x\) spełniające nierówność

\[\left(0,2 \right)^{4\, x+3} \geqslant 5 ^ {x+2}.\]

Rozwiązanie

\[\begin{aligned} \left(0,2 \right)^{4\, x+3} &\geqslant 5 ^ {x+2} \\[10pt] 5^{-1\left(4\, x+3\right)} &\geqslant 5 ^ {x+2} \\[10pt] \text{funkcja jest rosnąca, } &\text{zatem otrzymujemy}\\[10pt] -4 \, x -3 &\geqslant x + 2 \\[10pt] -5 &\geqslant 5\, x \\[10pt] -1 &\geqslant x \end{aligned}\]

Odpowiedź: \(x \in \left( \infty, -1 \right\rangle\).



Przykład 3.

Wyznacz \(x\) spełniające równanie

\[\left(\frac{3}{4} \right)^{x^3} = \left( \frac{4}{3}\right)^{x^2+x}. \]

Rozwiązanie

\[\begin{aligned} \left(\frac{3}{4} \right)^{x^3} &= \left( \frac{4}{3}\right)^{x^2+x} \\[10pt] \left(\frac{3}{4} \right)^{x^3} &= \left( \frac{3}{4}\right)^{-\left( x^2+x \right)} \\[10pt] \end{aligned}\] \[\begin{gather} \text{funkcja jest różnowartościowa, zatem otrzymujemy} \\[10pt] x^3 = -x^2 -x \\[10pt] x \left( x^2+x+1 \right) = 0 \\[10pt] x = 0 \end{gather}\]

Odpowiedź: \(x=0\).



Przykład 4.

Wyznacz \(x\) spełniające równanie

\[\left( 0,5 \right)^{x^2} \cdot 2^{2\, x+2} = \frac{1}{64}. \]

Rozwiązanie

\[\begin{aligned} \left( 0,5 \right)^{x^2} \cdot 2^{2\, x+2} &= \frac{1}{64} \\[10pt] 2^{-x^2+2\, x+2} &= 2^{-6} \\[10pt] \end{aligned}\] \[\begin{gather} \text{funkcja jest różnowartościowa, zatem otrzymujemy}\\[10pt] -x^2+2\, x+2=6 \\[10pt] x^2-2\, x-8=0\\[10pt] \left(x-4\right)\left(x+2\right) = 0 \\[10pt] x=4 \quad \vee \quad x=-2 \end{gather}\]

Odpowiedź: \(x=4 \vee x=-2\).



Przykład 5.

Wyznacz \(x\) spełniające nierówność

\[\left( \frac{2}{3} \right)^{x^3+6\, x^2} - \left( \frac{16}{81} \right)^{6\, x+16} \geqslant 0. \]

Rozwiązanie

\[\begin{aligned} \left( \frac{2}{3} \right)^{x^3+6\, x^2} &\geqslant \left( \frac{16}{81} \right)^{6\, x+16} \\[10pt] \left( \frac{2}{3} \right)^{x^3+6\, x^2} &\geqslant \left( \frac{2}{3} \right)^{4(6\, x+16)} \\[10pt] \end{aligned}\] \[\begin{gather} \text{funkcja jest malejąca, zatem otrzymujemy} \\[10pt] x^3+6\, x^2 \leqslant 24\, x+64 \\[10pt] x^3+6\, x^2-24\, x-64 \leqslant 0\\[10pt] x^2 \left(x+2\right)+4\, x\left(x+2\right)-32\left(x+2\right) \leqslant 0 \\[10pt] \left(x+2\right)\left(x^2+4\, x-32\right) \leqslant 0 \\[10pt] \left(x+2\right)\left(x+8\right)\left(x-4\right)\leqslant 0 \\[10pt] x \leqslant -8 \quad \vee \quad\left( x \geqslant -2 \quad \wedge \quad x \leqslant 4 \right) \end{gather}\]

Odpowiedź: \(x \in \left( -\infty, -8\right\rangle \cup \left\langle -2, 4\right\rangle\).



Zadanie 1.

Wyznacz \(x\) spełniające równanie

\[ \sqrt[6]{4^x \cdot 0,125^{\frac{1}{x}}} = \frac{4\sqrt[3]{2}}{\left(\sqrt{2}\right)^x}. \]

Rozwiązanie

\[\begin{gather} \sqrt[6]{4^x \cdot 0,125^{\frac{1}{x}}} = \frac{4\sqrt[3]{2}}{\left(\sqrt{2}\right)^x},\quad \mathbb{D} = x \in \mathbb{R} \setminus \left\{0\right\} \\[10pt] \sqrt[6]{2^{2\, x} \cdot 2^{-3\left(\frac{1}{x}\right)}} = \frac{2^\frac{7}{3}}{\left(2\right)^{\frac{1}{2}\, x}}\\[10pt] 2^{\frac{1}{6} \left( 2\, x -\frac{3}{x} \right)} = 2^{\frac{7}{2}-\frac{x}{2}} \\[10pt] \text{funkcja jest różnowartościowa, zatem otrzymujemy} \\[10pt] \frac{2\, x-\frac{3}{x}}{6}=\frac{7}{2}-\frac{x}{2} \\[10pt] 2\, x-\frac{3}{x} = 14 - 3\, x \\[10pt] 5\, x^2-14\, x -3 = 0 \\[10pt] \left(x-3\right)\left(5\, x+1\right)=0 \\[10pt] \left( x=3 \quad \vee \quad x=-\frac{1}{5} \right) \wedge x \in \mathbb{D} \end{gather}\]

Odpowiedź: \(x=3\vee x=-\frac{1}{5}\).



Zadanie 2.

Wyznacz \(x\) spełniające nierówność

\[ 3^{\frac{x}{x+2}}> \frac{3}{3^{\frac{1}{x-1}}}. \]

Rozwiązanie

\[\begin{gather} 3^{\frac{x}{x+2}}> \frac{3}{3^{\frac{1}{x-1}}},\quad \mathbb{D} = x \in \mathbb{R} \setminus \left\{-2, 1\right\} \\[10pt] 3^{\frac{x}{x+2}}> 3^{1 - \frac{1}{x-1}} \\[10pt] \text{funkcja jest rosnąca, zatem otrzymujemy} \\[10pt] \frac{x}{x+2}>1 - \frac{1}{x-1} \\[10pt] \frac{x}{x+2} - \frac{x-2}{x-1} >0 \\[10pt] \frac{x\left( x- 1\right)-\left( x- 2\right)\left( x+2\right)}{\left( x- 1\right)\left( x+2\right)}>0\\[10pt] \frac{x^2-x-x^2+4}{\left( x- 1\right)\left( x+2\right)} > 0\\[10pt] \text{badamy, dla jakich \(x\) funkcja przyjmuje wartości dodatnie} \\[10pt] \left( x+2\right)\left( x-1\right)\left( x- 4\right)<0 \\[10pt] \Bigl( x< -2 \quad \vee \quad \left(x>1 \quad \wedge \quad x< 4 \right) \Bigr) \quad\wedge\quad x \in \mathbb{D} \end{gather}\]

Odpowiedź: \(x \in \left(-\infty, -2\right)\cup \left( 1, 4\right)\).



Zadanie 3.

Wyznacz \(x\) spełniające równanie

\[ 4^{\sqrt{x-1}+1}-4^{\sqrt{x-1}-\frac{1}{2}}=7. \]

Rozwiązanie

\[\begin{gather} 4^{\sqrt{x-1}+1}-4^{\sqrt{x-1}-\frac{1}{2}}=7,\quad \mathbb{D}=\left\langle1, \infty\right) \\[10pt] 4^\sqrt{x-1}\cdot 4-4^\sqrt{x-1}\cdot\frac{1}{2}=7 \\[10pt] 4^\sqrt{x-1}\left( 4-\frac{1}{2}\right)=7 \\[10pt] 2^{2\sqrt{x-1}}=2^1 \\[10pt] \text{funkcja jest różnowartościowa, zatem otrzymujemy} \\[10pt] 2\sqrt{x-1}=1\\[10pt] \sqrt{x-1} = \frac{1}{2}\\[10pt] x-1 = \frac{1}{4}\\[10pt] x = \frac{5}{4}\quad\wedge\quad x \in \mathbb{D} \end{gather}\]

Odpowiedź: \(x = \frac{5}{4}\).



Zadanie 4.

Wyznacz \(x\) spełniające nierówność

\[ 3^{3\, x+1}-4\cdot 27^{x-1}+9^{1,5\, x-1}<80. \]

Rozwiązanie

\[\begin{gather} 3^{3\, x+1}-4\cdot 27^{x-1}+9^{1,5\, x-1} < 80 \\[10pt] 3^{3\, x+1}-4\cdot 3^{3\, x-3}+3^{2\, x-2} < 80 \\[10pt] 3^{3\, x}\left( 3 - \frac{4}{27}+\frac{1}{9}\right) < 80 \\[10pt] 3^{3\, x}\cdot\frac{80}{27} < 80 \\[10pt] 3^{3\, x} < 3^3 \\[10pt] \text{funkcja jest rosnąca, zatem otrzymujemy} \\[10pt] 3 \, x < 3\\[10pt] x < 1 \end{gather}\]

Odpowiedź: \(x \in \left( -\infty, 1\right)\).



Zadanie 5.

Wyznacz \(x\) spełniające równanie

\[ 2^{x^2-x+2}+5\cdot 2^{x^2-x-1}=26. \]

Rozwiązanie

\[\begin{gather} 2^{x^2-x+2}+5\cdot 2^{x^2-x-1}=26 \\[10pt] 2^{x^2-x+2}\left(1+ 5\cdot2^{-3}\right)=26 \\[10pt] 2^{x^2-x+2}\cdot \frac{13}{8}=26 \\[10pt] 2^{x^2-x+2} = 16 \\[10pt] 2^{x^2-x+2} = 2^4 \\[10pt] \text{funkcja jest różnowartościowa, zatem otrzymujemy} \\[10pt] x^2-x+2=4\\[10pt] x^2-x-2=0\\[10pt] \left(x-2\right)\left(x+1\right)=0\\[10pt] x = 2 \quad \vee \quad x = -1 \end{gather}\]

Odpowiedź: \(x = 2 \vee x = -1\).
