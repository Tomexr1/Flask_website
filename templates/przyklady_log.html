{% endif %}

Wyznacz \(x\) spełniające równanie

\[\log_4{\left(1+\log_3\left( 1+\log_2\left( x+3\right) \right)\right)}=\frac{1}{2}.\]

Rozwiązanie

\[\begin{gather} \log_4{\left(1+\log_3\left( 1+\log_2\left( x+3\right) \right)\right)}=\frac{1}{2} \\[10pt] \end{gather}\] w rozwiązaniu kilkakrotnie skorzystamy z definicji logarytmu \[\begin{gather} 4^{\frac{1}{2}} = 1+\log_3\left( 1+\log_2\left( x+3\right) \right) \\[10pt] 1 = \log_3\left( 1+\log_2\left( x+3\right) \right) \\[10pt] 3^1 = 1+\log_2\left( x+3\right) \\[10pt] 2 = \log_2\left( x+3\right) \\[10pt] 2^2 = x+3 \\[10pt] x = 1 \end{gather}\]

Odpowiedź: \(x = 1\).



Przykład 2. {% if current_user.is_authenticated %}
{% endif %}

Wyznacz \(x\) spełniające równanie

\[\log_\frac{1}{2}\log_8\frac{x^2-2\, x}{x-3}=0.\]

Rozwiązanie

\[\begin{gather} \log_\frac{1}{2}\log_8\frac{x^2-2\, x}{x-3}=0, \quad \mathbb{D}=\mathbb{R} \setminus \left\{3\right\}\\[10pt] \end{gather}\] w rozwiązaniu dwukrotnie skorzystamy z definicji logarytmu \[\begin{gather} \left(\frac{1}{2}\right)^0= \log_8\frac{x^2-2\, x}{x-3}\\[10pt] 8^1=\frac{x^2-2\, x}{x-3}\\[10pt] 8\, x -24 = x^2-2\, x \\[10pt] x^2-10\, x+24=0 \\[10pt] \left(x-4\right)\left(x-6\right) \\[10pt] \left( x = 4 \quad \vee \quad x=6 \right)\quad \wedge \quad x \in \mathbb{D} \end{gather}\]

Odpowiedź: \(x = 4 \vee x=6\).

{% endif %}

Wyznacz \(x\) spełniające nierówność

\[\log_2|x-3|\leqslant 1.\]

Rozwiązanie

\[\begin{gather} \log_2|x-3|\leqslant 1, \quad \mathbb{D}=\mathbb{R} \setminus \left\{3\right\}\\[10pt] \log_2|x-3|\leqslant \log_22 \\[10pt] \end{gather}\] funkcja jest rosnąca, zatem otrzymujemy \[\begin{gather} |x-3| \leqslant 2\\[10pt] \left( x \geqslant 1 \quad \wedge \quad x\leqslant 5 \right)\quad \wedge \quad x \in \mathbb{D} \end{gather}\]

Odpowiedź: \(x \in \left\langle 1,3\right)\cup \left(3, 5\right\rangle\).



Przykład 4. {% if current_user.is_authenticated %}
{% endif %}

Wyznacz \(x\) spełniające nierówność

\[\log_\frac{1}{2}\left(5\, x+10\right)< \log_\frac{1}{2}\left(x^2+6\, x +8\right).\]

Rozwiązanie

\[\begin{gather} x \in \mathbb{D} \Leftrightarrow \left(5\, x+10 > 0 \quad\wedge \quad x^2+6\, x+8 > 0 \right)\\[10pt] \log_\frac{1}{2}\left(5\, x+10\right)< \log_\frac{1}{2}\left(x^2+6\, x +8\right), \quad \mathbb{D}=\left(-2, \infty\right)\\[10pt] \end{gather}\] funkcja jest malejąca, zatem otrzymujemy \[\begin{gather} 5\, x+10 > x^2+6\, x +8\\[10pt] x^2+x-2 < 0\\[10pt] \left(x+2\right)\left(x-1\right) < 0\\[10pt] \left(x > -2 \quad \wedge \quad x< 1\right) \quad \wedge \quad x \in \mathbb{D} \end{gather}\]

Odpowiedź: \(\left(-2, 1\right)\).



Przykład 5. {% if current_user.is_authenticated %}
{% endif %}

Wyznacz \(x\) spełniające równanie

\[\log_{x-1}\left(2\, x^2+4\, x-6\right)=2.\]

Rozwiązanie

\(x\) występuje zarówno w podstawie, jak i w liczbie logarytmowanej,
dlatego wyznaczamy dziedzinę funkcji
\[\begin{gather} x \in \mathbb{D} \Leftrightarrow \left( 2\, x^2+4\, x-6> 0 \quad\wedge \quad x-1 > 0 \quad \wedge \quad x-1 \neq 1\right)\\[10pt] \log_{x-1}\left(2\, x^2+4\, x-6\right)=2, \quad \mathbb{D}=\left(1, 2\right)\cup \left(2, \infty\right)\\[10pt] \end{gather}\] korzystając z definicji otrzymujemy \[\begin{gather} \left(x-1\right)^2=2\, x^2+4\, x-6\\[10pt] x^2+6-7=0\, x\\[10pt] \left(x-1\right)\left(x+7\right) = 0\\[10pt] \left(x = -7 \quad \vee \quad x= 1\right) \quad \wedge \quad x \in \mathbb{D} \end{gather}\]

Odpowiedź: brak rozwiązań.

Wyznacz \(x\) spełniające równanie

\[ \log_{4-x^2}16=2. \]

Rozwiązanie(Pokaż)

\[\begin{gather} x \in \mathbb{D} \Leftrightarrow \left(4-x^2 > 0 \quad \wedge \quad 4-x^2 \neq 1\right)\\[10pt] \mathbb{D}=\left(-2, -\sqrt{3}\right)\cup\left(-2, -\sqrt{3}, \sqrt{3}\right)\cup\left(\sqrt{3}, 2\right)\\[10pt] \log_{4-x^2}16=2\\[10pt] 2 \log_{4-x^2}4=2\\[10pt] \log_{4-x^2}4=1\\[10pt] \end{gather}\] korzystając z definicji otrzymujemy \[\begin{gather} 4-x^2=4\\[10pt] x^2=0 \\[10pt] x=0 \quad \wedge \quad x \in \mathbb{D} \\[10pt] \end{gather}\]

Odpowiedź: \(x=0\).



Zadanie 2. {% if current_user.is_authenticated %}
{% endif %}

Wyznacz \(x\) spełniające równanie

\[ \log_3 \left(x-1\right)=\log_3\frac{1}{2\, x-1}. \]

Rozwiązanie(Pokaż)

\[\begin{gather} x \in \mathbb{D} \Leftrightarrow \left(x-1 > 0 \quad \wedge \quad \frac{1}{2\, x-1}> 0\quad\wedge\quad 2\, x-1 \neq 0\right)\\[10pt] \log_3 \left(x-1\right)=\log_3\frac{1}{2\, x-1}, \quad \mathbb{D}=\left(1, \infty\right)\\[10pt] \end{gather}\] funkcja jest różnowartościowa, zatem otrzymujemy \[\begin{gather} x-1=\frac{1}{2\, x-1}\\[10pt] \left(x-1\right)\left(2\, x-1\right)=1\\[10pt] 2\, x^2-x-2\, x +1-1=0\\[10pt] x\left(2\, x-3\right)=0\\[10pt] \left(x=0 \quad \vee \quad x=\frac{3}{2}\right)\quad \wedge \quad x \in \mathbb{D} \\[10pt] \end{gather}\]

Odpowiedź: \(x=\frac{3}{2}\).



Zadanie 3. {% if current_user.is_authenticated %}
{% endif %}

Wyznacz \(x\) spełniające nierówność

\[ \log_\frac{1}{4} \left(2-x\right)\geqslant\log_\frac{1}{4}\frac{2}{x+1}. \]

Rozwiązanie(Pokaż)

\[\begin{gather} x \in \mathbb{D} \Leftrightarrow \left(2-x > 0 \quad \wedge \quad \frac{2}{x+1}> 0\quad\wedge\quad x+1 \neq 0\right)\\[10pt] \log_\frac{1}{4} \left(2-x\right)\geqslant\log_\frac{1}{4}\frac{2}{x+1}, \quad \mathbb{D}=\left(-1, 2\right)\\[10pt] \end{gather}\] funkcja jest malejąca, zatem otrzymujemy \[\begin{gather} 2-x\leqslant\frac{2}{x+1}\\[10pt] \frac{\left(2-x\right)\left(x+1\right)-2}{x+1}\leqslant 0\\[10pt] \frac{-x^2+x}{x+1}\leqslant 0\\[10pt] \end{gather}\] badamy, dla jakich \(x\) funkcja przyjmuje wartości ujemne \[\begin{gather} -x\left(x-1\right)\left(x+1\right)\leqslant 0\\[10pt] \bigl(\left(x\geq 1 \quad \wedge \quad x\leqslant 0\right)\quad \vee \quad x\geqslant 1\bigr)\quad \wedge \quad x \in \mathbb{D} \\[10pt] \end{gather}\]

Odpowiedź: \(x \in \left( -1, 0\right\rangle \cup \left\langle 1, 2\right) \).



Zadanie 4. {% if current_user.is_authenticated %}
{% endif %}

Wyznacz \(x\) spełniające równanie

\[ \log_4\sqrt{x}+\frac{1}{2}\log_4\left(x+4\right)=\frac{5}{4}. \]

Rozwiązanie(Pokaż)

\[\begin{gather} \log_4\sqrt{x}+\frac{1}{2}\log_4\left(x+4\right)=\frac{5}{4}, \quad \mathbb{D}=\left(0, \infty\right)\\[10pt] \frac{1}{2}\left(\log_4x+\log_4\left(x+4\right)\right)=\frac{5}{4}\\[10pt] \end{gather}\] funkcja jest różnowartościowa, zatem otrzymujemy \[\begin{gather} 4^{\frac{10}{4}}=x\left(x+4\right)\\[10pt] x^2+4\, x-2^5=0\\[10pt] \left(x+8\right)\left(x-4\right)=0 \\[10pt] \left(x=4 \quad \vee \quad x=-8\right)\quad \wedge\quad x \in \mathbb{D}\\[10pt] \end{gather}\]

Odpowiedź: \(x = 4 \).



Zadanie 5. {% if current_user.is_authenticated %}
{% endif %}

Wyznacz \(x\) spełniające równanie

\[ \log\left(x-1\right)+\log\left(x+1\right)=\log 8+\log\left(x-2\right). \]

Rozwiązanie(Pokaż)

\[\begin{gather} x \in \mathbb{D} \Leftrightarrow \left(x-1 > 0 \quad \wedge \quad x+1 > 0\quad\wedge\quad x-2 > 0\right)\\[10pt] \log\left(x-1\right)+\log\left(x+1\right)=\log 8+\log\left(x-2\right), \quad\mathbb{D}=\left(2, \infty\right)\\[10pt] \log\left(x^2-1\right)=\log\left(8\left(x-2\right)\right)\\[10pt] \end{gather}\] funkcja jest różnowartościowa, zatem otrzymujemy \[\begin{gather} x^2-1=8\, x-16\\[10pt] x^2-8\, x+15=0\\[10pt] \left(x-3\right)\left(x-5\right)=0 \\[10pt] \left(x=3 \quad \vee \quad x=5\right)\quad \wedge\quad x \in \mathbb{D}\\[10pt] \end{gather}\]

Odpowiedź: \(x = 3 \vee x=5\).
